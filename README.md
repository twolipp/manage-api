# Manager Api

## Backend Setup:

**Directory:** backend
**Language:** Python 3
**Framework:** Django

### Steps:

- **Navigate:** `cd backend`
- **Create Virtual Environment:** `virtualenv venv`
- **Activate Virtual Environment:** `source venv/bin/activate`
- **Install Dependencies:** `pip install -r requirements.txt`
- **Migrate Database:** `cd management && python manage.py migrate`
- **Run Development Server:** `python manage.py runserver`
- **Access GraphQL Interface:** Visit http://localhost:8000/graphql in your browser.

## Frontend Setup:

**Directory:** frontend/app
**Library:** React

### Steps:

- **Navigate:** `cd frontend/app`
- **Install Dependencies:** `yarn install`
- **Run Development Server:** `yarn start`
- **Access React App:** Visit http://localhost:3000 in your browser.
